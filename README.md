# 🏍️ GVS Sport - Institucional & Catálogo

Landing page institucional desenvolvida para a **GVS Sport**, líder na fabricação de retrovisores e acessórios para motocicletas. O foco deste projeto foi a criação de um layout profissional, responsivo e com alta fidelidade à identidade visual da marca.

## 🛠 Tecnologias Utilizadas
- **HTML5:** Semântica voltada para clareza e estrutura do conteúdo.
- **Tailwind CSS:** Framework utilitário para estilização rápida, responsiva e otimizada.
- **JavaScript (ES6):** Implementação de navegação suave (*smooth scroll*) personalizada e configuração de temas no Tailwind.
- **Google Fonts:** Tipografia 'Inter' para uma leitura limpa e profissional.

## 💡 Principais Funcionalidades
- **Design Responsivo:** Layout adaptado para dispositivos móveis e desktops utilizando o sistema de *breakpoints* do Tailwind.
- **Navegação Inteligente:** Sistema de *scroll* suave personalizado para melhor usabilidade entre as seções.
- **Identidade Visual:** Customização de paleta de cores (Verde GVS e Dark) integrada ao *tailwind.config*.
- **Interatividade:** Efeitos de escala (*hover effects*) em cards e botões, melhorando a experiência do usuário.

## 🎓 Aprendizados
Este projeto permitiu aprofundar conhecimentos em:
- **Tailwind CSS:** Compreensão de como criar temas customizados e escalar estilos sem a necessidade de múltiplos arquivos CSS.
- **UX/UI:** Aplicação de *best practices* de design, como o uso de *backdrop-filter* (efeito glassmorphism) e hierarquia visual.
- **Configuração de ambiente:** Manipulação do DOM para criar componentes que respondem dinamicamente ao usuário.
